Se utiliza python con las librerias Request, json y Pandas para el manejo de datos


Mediante la Libería Pandas se logra crear un DataFrame que lea los datos en formato Json y los convierta en datos en una tabla legible


Mediante la funcion Shape podemos obtener las dimensiones del DataFrame con el formato de filas, columnas. En nuestro caso al aplicar la funcion Shape (dataframe.shape) nos genera el siguiente resultado:


{{ dimensiones }}


Con esto nos damos por enterados que el DataFrame posee 49 filas y 7 columnas, ademas podemos obtener una información básica del como esta constituído ese dataFrame mediante la función info(DataFrame.info())


{{ info }}


Como vemos la informacion que nos devuelve la funcion puede ser un poco confusa y compleja para personas no acostumbradas a lidiar con los bufer y con las salidas en pantalla que proporciona Python, afortunadamente tenemos otras funciones dentro de la librería que nos permiten realizar ciertas búsquedas de información por ejemplo tenemos una funcion que nos permiten hacer un listado de los nombres de las columnas de nuestro dataFrame (DataFrame.columns.values) lo cual nos genera la siguiente salida:
